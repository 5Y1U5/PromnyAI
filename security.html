Security

セキュリティへの取り組み

お客様の大切なデータを安全に保管・処理するために、
複数のセキュリティ対策を実施しています。
法人としても安心してチーム全体でご利用いただける環境です。

lock

通信の暗号化

すべての通信はTLS（SSL）により暗号化されています。ブラウザ・サーバー間、サーバー・データベース間のデータ転送は第三者に傍受されることなく安全に行われます。

dns

データ保管

データはGoogle Cloud 東京リージョン（asia-northeast1）に保管されます。保存データはAES-256で暗号化され、転送時もTLSで保護。Google Cloudの多層暗号化アーキテクチャにより、高いセキュリティ基準で管理されています。

shield

テナント分離

企業ごとにデータは完全に分離されています。他企業のチャット履歴や生成データへのアクセスは、システム設計上不可能です。すべてのクエリにテナントIDフィルタを適用しています。

visibility_off

AI処理のプライバシー

お客様のデータはAIモデルの学習用に利用されることはありません。AIへの送信は応答生成の目的のみに限定されており、処理後のデータはモデル側に保持されません。

delete_sweep

データ削除

アカウント削除時には、チャット履歴・生成データ・ファイルなどすべてのデータを完全に削除します。バックアップデータも所定期間内に消去されます。

admin_panel_settings

認証・アクセス制御

パスワードはハッシュ化して保管。管理画面へのアクセスはメール認証により保護されています。管理者とユーザーの権限を分離し、適切なアクセス制御を実現しています。

Infrastructure

利用インフラの安全性

PromnyAIが利用する外部サービスは、いずれも国際的なセキュリティ基準を満たしています。

G

Google Cloud

サーバーインフラ

  • check SOC 1 / SOC 2 Type II / SOC 3 認証取得
  • check ISO/IEC 27001 / 27017 / 27018 / 27701 認証取得
  • check 東京リージョン（asia-northeast1）でデータ保管
  • check 保存時AES-256暗号化・転送時TLS暗号化がデフォルト適用
D

Dify

AIアプリケーション基盤

  • check SOC 2 Type I / Type II 認証取得
  • check ISO 27001:2022 認証取得
  • check GDPR準拠・ロールベースのアクセス制御
  • check 全アクセス履歴のログ記録・自動バックアップ
S

Stripe

決済システム

  • check PCI DSS Level 1認定（決済業界最高レベル）
  • check SOC 1 / SOC 2 Type II 認証取得、ISO 27001認証取得
  • check カード情報の非保持化（Stripe Elementsによるトークナイゼーション）
  • check カード番号はAES-256で暗号化、復号鍵は隔離された専用サーバーで管理
  • check 3Dセキュア 2.0対応（日本の規制要件に準拠）
smart_toy

LLMプロバイダー

OpenAI / Anthropic / Google

  • check 全プロバイダーがSOC 2 Type II認証取得
  • check API経由のデータはモデル学習に不使用
  • check OpenAI・Anthropic：ISO 27001認証取得
  • check 処理後のデータはプロバイダー側に保持されません

ご注意：本ページに記載のセキュリティ認証（SOC 2、ISO 27001、PCI DSS等）は、各サービスプロバイダーが取得・準拠しているものであり、PromnyAIが直接取得したものではありません。PromnyAIはこれらの認証を取得したインフラ上で運用されることで、高いセキュリティ水準を実現しています。各認証の詳細は、各サービスの公式セキュリティページをご確認ください。
