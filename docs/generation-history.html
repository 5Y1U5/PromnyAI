生成履歴の活用

過去に作った文章を見返し、似たテーマの発信に再利用する

このマニュアルでできること

生成履歴は、PromnyAIのコンテンツ生成で作ったブログ・SNS・LINE配信文をあとから見返すための場所です。過去の文章を探し、似たテーマの発信に再利用し、表現の揺れを減らせます。

ここでは、履歴の開き方、検索・絞り込み、再利用の判断、チームで使うときの注意点を整理します。コンテンツ生成の基本操作は、先にコンテンツ生成の使い方を確認してください。

生成履歴を開く

  1. PromnyAIにログインする
  2. サイドバーから「コンテンツ生成」を開く
  3. 画面上部、またはサイドバー内の「履歴」を開く
  4. 過去に作成したコンテンツの一覧を確認する

履歴には、作成した日時、コンテンツ種別、テーマ、本文が残ります。似た内容をもう一度作るときは、新しくゼロから入力する前に履歴を確認してください。

履歴で確認する項目

  • schedule作成日時：いつ作った文章かを確認します。キャンペーンや価格が変わる業務では特に重要です。
  • categoryコンテンツ種別：ブログ、Instagram、X、Facebook、LINEなど、媒体を確認します。
  • titleテーマ：何について生成した文章かを確認します。
  • article本文：過去の表現、見出し、告知文の言い回しを確認します。

過去文章を再利用する手順

過去文章は、そのままコピーするより「型」として使うと安定します。次の流れで再利用してください。

  1. 履歴から似たテーマの文章を探す
  2. 残したい構成、見出し、言い回しを確認する
  3. 新しいテーマ、日付、商品名、金額を別メモに書き出す
  4. コンテンツ生成の「参考テキスト」に過去文章を貼る
  5. 追加指示に「構成だけ参考にし、事実は新しい情報を使う」と書く
  6. 生成後、古い日付・価格・キャンペーン名が残っていないか確認する

再利用してよい文章、避けたい文章

履歴に残っている文章なら何でも再利用してよいわけではありません。次の基準で選んでください。

  • check再利用しやすい：反応が良かった投稿、社内で直しが少なかった文章、季節を問わない説明文。
  • check一部だけ使う：キャンペーン文、イベント告知、期間限定の案内。構成だけ参考にします。
  • warning避けたい：古い価格、終了済み特典、在庫状況、担当者名が入った文章。

おすすめの追加指示

以下の過去文章を参考に、新しい告知文を作ってください。

参考にするのは構成、文体、見出しの流れだけです。日付、価格、商品名、キャンペーン内容は今回の情報を優先してください。

古い情報が残りそうな箇所は、本文に入れず「確認が必要」と書いてください。

この一文を入れるだけで、過去文章の丸写しを避けやすくなります。特にLINE配信やSNS投稿では、季節・在庫・価格の確認を必ず行ってください。

チームで履歴を使うときのルール

複数人で使う場合は、履歴を「過去の答え置き場」ではなく「表現の見本」として扱います。次の3つを決めておくと運用しやすくなります。

  1. 採用基準：反応が良かった投稿、上長確認済みの文章だけを参考にする
  2. 確認担当：外部公開前に、数字・日付・名前・金額を確認する人を決める
  3. 更新日：月1回、古いキャンペーンや終了済み告知を確認する

注意点

  • warning履歴の文章は、生成当時の情報をもとにしています。公開前に必ず最新情報と照合してください。
  • warning個人情報、顧客名、未公開情報が含まれる文章は、社内ルールに沿って扱ってください。
  • warning過去文章をそのまま繰り返すと、読者には機械的に見えることがあります。毎回、今回の出来事や日付を入れてください。
