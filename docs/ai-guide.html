AI Guide

AI選び方ガイド

やりたいこと別ガイド

どのAIを使えばいいか迷ったら、やりたいことから選べます。PromnyAIでは全22種類のモデルをすぐに切り替えて使えます。

文章を書きたい → Claude

日本語が自然で文章作成に強いAIです。ブログ記事、メール返信、キャッチコピーなど、質の高い文章が必要なときに最適です。

  • smart_toyClaude Opus 4.6 — 最高性能。複雑な文章や長文コンテンツに
  • smart_toyClaude Sonnet 4.6 — バランス型。日常的な文章作成に
  • smart_toyClaude Haiku 4.5 — 高速・軽量。短文やシンプルな返信に

調べものをしたい → Perplexity

最新情報をネットから検索し、出典付きで回答してくれる検索特化AIです。トレンド調査や競合リサーチに便利で、信頼性の高い情報を素早く収集できます。

  • checkリアルタイムのネット検索に対応
  • check出典（URL）付きで回答するため信頼性が高い
  • check業界トレンドや競合情報の収集に最適

画像やPDFを分析したい → Gemini

PDF読み込みや画像の分析が得意なAIです。資料の要約や情報抽出、図解の読み取りなどに活用できます。

  • smart_toyGemini 3.1 Pro Preview — 最新モデル。高精度な分析に
  • smart_toyGemini 3 Flash Preview — 高速処理。素早い要約に
  • smart_toyGemini 2.5 Pro — 安定した定番モデル

とりあえず相談したい → ChatGPT

何でもバランスよくこなせる万能タイプです。「何から始めればいいかわからない」という場合はまずChatGPTを試してみてください。

  • smart_toyChatGPT 5.4 — 最新・最高性能モデル
  • smart_toyChatGPT 5.3 — 安定した高性能モデル
  • smart_toyChatGPT 5 mini — 軽量・高速。シンプルな作業に

画像を作りたい → 画像生成AI

チラシやSNS投稿用のPOP画像、バナーなどをテキスト指示から生成できます。外注不要で、自分で即作成できます。

  • smart_toyDALL-3 — OpenAIの定番画像生成モデル
  • smart_toygpt-image-1 — 高品質・高精度な最新画像生成モデル

業務を効率化したい → 業界特化AI

PromnyAIに組み込まれた業界特化AIチャットボットを使えば、プロンプトを考える手間なく即座に業務直結の出力が得られます。選ぶだけで完了です。

  • checkブログ記事の自動作成 → BlogBot_改
  • check口コミ返信文の自動生成 → レビュージェネレーター
  • checkLINE配信文の自動作成 → LINE配信文生成
  • checkメール返信文の自動作成 → ビジネスメール返信文生成
  • check経営・ビジネス相談 → CriticalCatalyst
